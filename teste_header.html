Teste do Header

Se você consegue ver o header acima com o logo "Caiu o Servidor" e o menu hambúrguer, então está funcionando corretamente.

O header deve ter:

  • Fundo escuro
  • Logo e título em branco
  • Botão hambúrguer no canto esquerdo
  • Previsão do tempo no canto direito
  • Menu lateral que abre ao clicar no hambúrguer

Teste do Menu Lateral

Clique no botão hambúrguer (☰) no canto esquerdo do header para abrir o menu lateral.

O menu deve conter links para:

  • Início
  • Painel de Usuário
  • Lista de Usuários
  • Contato
  • Cadastrar Anúncio
  • Sair
